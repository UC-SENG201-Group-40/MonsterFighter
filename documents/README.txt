Using the CLI:

- "./gradlew build" to build the project
- "./gradlew run" to run the project

Import to Eclipse:

- Go to "File" menu
- Select "Import ..."
- Choose "Gradle Project"
- Select project root directory, (will be this directory, not just "/app")
- Go to "Run" menu, select "Run" action, and select the "Java Application" (for running the ppplication)

Folder structure:
- "./javadoc" Compiled Javadoc
- "./app" Application source code excluding build settings
- "./diagrams" UML diagrams (use case and class diagram)
- "./report" Report pdf file
- "./jar" Compiled jar application
